
# Arquitetura do Sistema (Produto Interno)

Frontend: Next.js + Tailwind + shadcn/ui
Backend: NestJS (Node.js) - Monólito modular
Banco: PostgreSQL
Realtime: Socket.io
Storage: Google Drive (Shared Drive)
Agenda: Google Calendar
Deploy: Cloud Run / Render / Railway

Fluxo:
Equipe → Web App → API → PostgreSQL → Google Drive / Google Calendar
